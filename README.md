# GameOfLifeOnCube

## 概要

[![thumbnail](https://pbs.twimg.com/ext_tw_video_thumb/1339214657886670848/pu/img/_nJ1LpRMRww1XDP5.jpg)(https://twitter.com/e_perulatus/status/1339214711989043200/video/1)

## 操作方法

* ボタン A: 選択
* ボタン B: 決定
* ボタン C: キャンセル
* タッチ操作: キューブの回転

## 動作環境

M5Core2 でのみ動作確認しています。
